TypeScript 学习

2023年11月16日 TypeScipt

TS是一个可选的静态的类型系统

如何进行类型约束

通过在变量、函数参数、函数返回值 位置加上变量名:类型进行标注

添加类型约束后，再进行赋时name=2214 ts编译器会进行类型检查，变量要求为 string 类型，但接收到的为number，当检查出错误时他会提示报错

基本类型

  1. number

  2. string

  3. boolean

  4. array

  5. null 和 undefined

    undefined 和 null 可以作为其他类型的子类型，他们可以赋值给其他类型

    在tsconfig.json中 添加strictNullChecks:true,可以使用更严格的空类型检查，启用后 null 和 undefined 只能赋值给自身。

  6. object

其他类型

  1. 联合类型
    多种类型的联合，满足任意一种类型即可

  2. void
    表示没有返回值。当函数没有明确指定返回值类型或者返回类型被指定为void时，该函数被认为不返回任何值。

  3. never
    表示永远不会发生赋值的类型

  4. 字面量类型
    使用一个值进行约束

  5. any类型
    任意类型，any类型可以绕过类型检查，因此any类型的数据可以赋值给任意类型

  6. 元祖类型
    固定长度，并且数组的每一项类型都确定,通常被用作返回值
